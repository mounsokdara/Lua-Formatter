# **App Name**: LuaForge

## Core Features:

- Code Input: Provide a text area for users to input or paste their Lua code.
- Beautify Code: Automatically format and indent the Lua code to improve readability.
- Delete All Comments: Provide a feature to delete all the single and multi line comments present in Lua code
- Ask User to delete comments or convert it into --[[]]: Ask User to delete comments or convert it into --[[]] when using beautifier
- Basic Text Editor: Basic Text Editor
- Download/Copy Output: Allow users to download the beautified or comment-free code as a .lua file.
- Muti line converter: Add Muti line converter into one liner convert by removing All Down spade <br> and delete all space but keep 1 spacebar minimum

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to reflect coding and structure.
- Background color: Light gray (#F0F2F5), near white, for easy readability.
- Accent color: Teal (#009688) for buttons and important interactive elements.
- Body and headline font: 'Inter', sans-serif, to ensure a clear and modern interface.
- Code font: 'Source Code Pro' for the display of code.
- A clean, centered layout with sufficient padding around all elements. The input area will occupy a majority of the page.
